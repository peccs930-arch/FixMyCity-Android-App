## Project Description

FixMyCity is an Android application that allows citizens to report public issues such as potholes, garbage dumps, water leaks, traffic issues, and broken streetlights.

## Features

* User Authentication
* GPS Location Tracking
* Google Maps Integration
* Issue Reporting
* Image Upload
* Complaint Tracking
* Push Notifications

## Technology Stack

* Kotlin
* Android Studio
* Firebase Authentication
* Firebase Firestore
* Google Maps API
* MVVM Architecture
